# uTag

uTag is easy way to change mp3 file tags

## Installation

Use shell prompt (bash, zsh, etc) for installation!
Check command below:

```bash
sh build.sh
```

## Usage

Use shell prompt (bash, zsh, etc) for execution!
Check command below:

```bash
./utag [-f <dir>]
```
